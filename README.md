# 科研日报阅读页

打开同一目录中的 `index.html`，即可离线阅读日报；不需要安装软件或启动服务。日报数据按月保存在 `reports` 文件夹中。

## 新增一天日报

1. 用文本编辑器打开 `reports` 文件夹中对应月份的文件，例如 `2026-07.js`。
2. 复制文件末尾的“新增日报模板”，粘贴到 `window.RESEARCH_REPORTS.push(...[ ])` 的数组中。
3. 填写日期、今日简报（内容总结与趋势判断）、Top 2 的论文 id 和每篇论文的点评内容。
4. 保存后重新打开或刷新 `index.html`。

日期使用 `YYYY-MM-DD` 格式，页面会自动把最新日期显示在最上方。同一天只能保留一条日报；若重复，页面只显示数组中靠前的那条。

“今日简报”中的推荐论文数和 A / B / C 优先级数量由页面按当天论文自动统计，无需手填。

## 跨月新增文件

进入新月份时，在 `reports` 文件夹中新建 `YYYY-MM.js`，沿用已有月度文件的结构；然后在 `reports/manifest.js` 的月份数组末尾加入该月份。页面会按清单自动读取所有月度文件。

## 出现提示时怎么办

- “无法读取日报数据”：确认 `reports/manifest.js` 存在，且清单中每个对应的月度文件都存在。
- “数据格式不正确”：确认 `window.RESEARCH_REPORTS` 是由方括号包起来的日报列表。
- `papers` 或 `topPicks` 提示错误：确认它们都使用方括号；Top 2 中的 id 也必须出现在当天论文的 `id` 中。

未核实的信息可以先写“待补充”，页面会正常显示。
